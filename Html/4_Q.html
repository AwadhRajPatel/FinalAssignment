  1. <html>: The root element of an HTML document, indicating the start of the document.
  2. <head>: Contains metadata about the HTML document, such as title, links to stylesheets, and scripts.
  3. <body>: Contains the visible content of the HTML document, including text, images, and other elements.
  4. <h1> - <h6>: Heading elements used to structure the content with different levels of importance.
  5. <p>: Paragraph element used to display text in a block format.
  6. <a>: Anchor element used to create links to other web pages or resources.
  7. <img>: Image element used to embed images into the HTML document.
  8. <ul>: Unordered list element used to create lists with bullet points.
  9. <ol>: Ordered list element used to create lists with numbers.
  10. <li>: List item element used to define individual items in a list.
